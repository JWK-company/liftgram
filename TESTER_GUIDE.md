# Liftgram POC — 테스터 가이드 (동업자 3인용)

운동 기록(Hevy 스타일) + 소셜 피드(인스타 스타일)를 합친 앱입니다.
아래 순서로 5~10분만 만져보면 전체 그림이 잡힙니다.

---

## 접속
1. 공유받은 링크를 **폰 브라우저(크롬/사파리)** 로 엽니다: `https://<배포주소>`
2. **홈 화면에 추가** → 앱처럼 아이콘으로 실행됩니다.
   - iPhone: 공유 버튼 → "홈 화면에 추가"
   - Android: 메뉴(⋮) → "홈 화면에 추가" / "앱 설치"

> ⏳ **첫 접속이 40~60초 느릴 수 있어요.** 무료 서버가 잠들어 있다가 깨어나는 시간입니다.
> "서버가 느리다"는 안내가 뜨면 잠깐 기다렸다 다시 시도하면 됩니다. (그 후엔 정상 속도)

## 로그인 (비밀번호 찾기 기능이 아직 없어요)
- 프로필 탭 → **로그인/가입** → 이메일·비밀번호로 가입.
- 셋이서 **각자 기억하기 쉬운 계정**으로 가입하거나, 아래처럼 미리 정해두면 편합니다:
  - 예) `alice@test.com` / `bob@test.com` / `carol@test.com`, 비번은 공통 하나.
- ⚠️ 비번을 잊으면 복구가 안 되니, 정한 계정·비번은 메모해 두세요.

---

## 이것만 해보면 됩니다 (체크리스트)

**운동 기록 (인터넷 없어도 됨 · Hevy 파트)**
- [ ] 루틴 만들기 → 운동 종목 몇 개 추가 → 순서 드래그
- [ ] 운동 시작 → 세트별 무게·횟수 입력 → 세트 완료(체크) → 휴식 타이머 확인
- [ ] 운동 완료 → 요약 확인 → 기록/분석 탭에서 볼륨·추정 1RM 보기

**소셜 (로그인 필요 · 인스타 파트)**
- [ ] 오운완/글/사진을 피드에 공유
- [ ] 서로 **팔로우** (발견 탭 → 사람 찾기)
- [ ] 좋아요 · 댓글 · 대댓글 · 댓글 좋아요
- [ ] **DM** 보내기 (1:1, 그룹도 가능) — 실시간으로 오는지
- [ ] 게시물 **저장(북마크)** → 프로필 → "저장한 게시물"에서 확인
- [ ] 발견/검색(사람·해시태그·글), 스토리(24시간), 신고/차단

**계정/기록 백업**
- [ ] 로그인하면 운동 기록이 서버로 자동 백업됩니다(다른 기기에서 같은 계정으로 로그인하면 이어짐).

## 알아두면 좋은 것 (POC 한계)
- **첫 접속 느림**: 위에 적은 콜드 스타트. 정상입니다.
- **사진**: 서버가 재배포되면 올린 사진이 사라질 수 있어요(무료 저장 특성). 글·계정·기록은 안전.
  (Cloudflare R2 연동 시 사진도 영구 — DEPLOY.md 참고)
- **아직 없는 것**: 비밀번호 찾기, 결제/구독, 자동 카운팅. POC엔 불필요.

## 문제가 생기면
- 화면이 계속 로딩만 돌면: 새로고침(당겨서) 또는 잠시 후 재시도(서버 깨우는 중).
- 이상한 점·좋은 점 뭐든 캡처해서 공유해 주세요 — 그게 이 POC의 목적입니다. 🙌
