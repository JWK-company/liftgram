# SAD-012 · 미디어 파이프라인·콘텐츠 모더레이션 (업로드·스토리지·CDN·트랜스코딩·스토리 만료)

## 개요 (Summary)
미디어 게시물·스토리(SRS-019)와 공개 프로필 미디어(SRS-008)를 위한 **업로드 → 저장 → 전송 → 모더레이션** 파이프라인이다. 사진/영상의 업로드, 오브젝트 스토리지, CDN 전송, 영상 트랜스코딩/썸네일, **스토리 24h 만료(TTL)**, **콘텐츠 모더레이션**(자동 + 신고 큐)을 담당한다. 저장·CDN·트랜스코딩 스택은 ADR-016, 모더레이션·안전·개인정보(민감정보·미성년) 정책은 ADR-017에 따른다. 소셜 노출·피드 합성은 SAD-011이 담당하며 본 파이프라인은 미디어 자산과 그 상태만 책임진다.

## 컴포넌트 / 책임
- **Upload Service**: 클라이언트 직접 업로드(서명 URL)·형식/크기 검증·메타데이터 기록(SRS-019).
- **Object Storage & CDN**: 원본·파생물 저장, CDN 캐시 전송·접근제어(공개범위 반영, ADR-016).
- **Transcoding & Thumbnail**: 영상 트랜스코딩·해상도 변형·썸네일/프리뷰 생성(SRS-019).
- **Story Lifecycle**: 스토리 24h TTL·자동 만료/비노출·자산 정리(SRS-019).
- **Moderation Pipeline**: 업로드 시 자동 필터(해시/분류) + 신고 큐·검수·내림/서스펜드, 민감정보·미성년 정책 집행(ADR-017).

## 주요 결정 포인트 (→ ADR 후보)
- 미디어 저장·CDN·트랜스코딩 스택(예: 오브젝트 스토리지 + 매니지드 트랜스코딩 + CDN) → ADR-016.
- 콘텐츠 모더레이션·안전·개인정보(건강=민감정보·미성년) 정책 → ADR-017.

## 연결
- 다루는 SRS: SRS-019(미디어 게시물·스토리), SRS-008(공개 프로필 미디어)
- 소셜 노출·피드·랭킹: SAD-011
