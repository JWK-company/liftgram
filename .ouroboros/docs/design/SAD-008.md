# SAD-008 · 한국 로컬라이즈 & i18n/콘텐츠

## 개요 (Summary)
로컬라이즈 계층은 **한국어 UI(i18n)와 한국어 운동명/별칭 DB**, **헬스장 가용 기구 기반 추천**, **오운완 인증 문화 요소**를 제공한다(SRS-013). 운동 카탈로그(SAD-002)에 한국어 별칭·한국 헬스장 기구 분류를 확장하고, 소셜 루프(SAD-005)에 오운완 인증을 1탭 게시 흐름으로 연결한다. i18n 구조는 추후 다국어(예: 일본) 확장을 막지 않도록 키 기반으로 설계한다.

## 컴포넌트 / 책임
- **i18n Framework**: 키 기반 다국어 리소스, 한국어 1차 완전 지원, 날짜/단위 현지화.
- **Korean Exercise DB Extension**: 한국어 운동명·별칭, 한국 헬스장 기구 분류를 카탈로그에 매핑(SRS-013).
- **Equipment-based Recommender**: 사용자 지정 가용 기구로 수행 가능한 루틴/대체 운동만 필터링(SRS-013, SAD-006 연계).
- **오운완 Certification Flow**: 운동 완료 인증을 1탭 게시 → 피드/그룹 노출(SAD-005 연계).

## 주요 결정 포인트 (→ ADR 후보)
- 한국어 운동 DB 구축 출처/저작권 정책 → ADR.
- 다국어 확장 로드맵(키 구조 vs 하드코딩) → ADR.
