# SAD-003 · 분석 엔진 (볼륨·추정1RM·PR·추세)

## 개요 (Summary)
분석 엔진은 완료된 세트 로그를 입력받아 **볼륨·추정 1RM·PR·부위별 분포·기간 추세**로 환산하는 결정적 집계 계층이다. 추정 1RM은 문서화된 표준 산식(Epley 등)을 사용해 동일 입력에 일관된 값을 산출하고, 모든 수치는 과장 없이 사실 기반으로 제시된다(웰니스, SAD-010 가드레일 준수). 집계는 로컬에서 즉시 갱신되며 무거운 추세 계산은 캐시/백그라운드로 분리한다.

## 컴포넌트 / 책임
- **Metric Calculator**: 세트→볼륨, 추정 1RM(표준 산식), PR 판정(최대중량/최대볼륨/최대반복)을 결정적으로 계산(SRS-005).
- **PR Detector**: 세션 종료 시 신규 PR을 즉시 감지해 요약/알림에 플래그.
- **Trend Aggregator**: 주/월/전체 기간 볼륨·세션수·부위분포 시계열 집계 및 캐싱.
- **Drilldown Provider**: 종목 단위 1RM 추세·PR 이력 조회 및 원천 세션으로의 링크.

## 주요 결정 포인트 (→ ADR 후보)
- 추정 1RM 산식 표준 선택(Epley vs Brzycki 등)과 표기 정책 → ADR.
