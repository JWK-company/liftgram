# SAD-007 · 자동 횟수 카운팅 (온디바이스 추론·수동폴백)

## 개요 (Summary)
자동 카운팅은 **온디바이스 센서/카메라 추론**으로 지원 종목의 반복 횟수를 집계해 세트 로깅 마찰을 제거한다(SRS-012). 프라이버시·지연·오프라인을 위해 추론은 기본 **온디바이스**(예: 모바일 비전 파이프라인)로 수행하고 원시 영상은 저장·전송하지 않는다. 정확도가 검증된 소수 종목에서 시작해 점진 확대하며, 미지원·실패·권한 거부 시 **수동 입력으로 무중단 폴백**한다. 본 기능은 횟수 카운팅에 한정하고 자세 평가·부상 진단을 표방하지 않는다(웰니스, SAD-010).

## 컴포넌트 / 책임
- **Capture & Inference**: 센서/카메라 입력에서 반복 동작을 온디바이스로 인식·카운트(원시 데이터 비저장).
- **Confidence & Correction**: 인식 신뢰도 평가, 낮으면 경고, 저장 전 1탭 ± 보정(SRS-012).
- **Fallback Controller**: 미지원 종목·권한 거부·실패 감지 시 수동 입력 모드로 즉시 전환.
- **Supported-Exercise Registry**: 정확도 검증된 지원 종목 목록과 점진 확대 관리.

## 주요 결정 포인트 (→ ADR 후보)
- 추론 위치(온디바이스 vs 서버)와 프라이버시·정확도 트레이드오프 → ADR.
- 초기 지원 종목 범위 협소화 정책(부상 책임·정확도) → ADR.
