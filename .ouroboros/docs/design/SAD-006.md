# SAD-006 · AI 프로그래밍/주기화 서비스 (생성·점진과부하·디로드)

## 개요 (Summary)
AI 프로그래밍 서비스는 Hevy가 안 하는 "프로그램을 짜주는 코치 레이어"로, 사용자 입력(목표·경력·장비·일수)을 받아 **다주차 구조화 프로그램을 생성**하고, 수행 데이터에 따라 **주차별 자동 진행/디로드**한다(SRS-009/010). 안전과 일관성을 위해 자유 생성이 아닌 **규칙 엔진 + 템플릿 + (선택) LLM 보조** 하이브리드로 설계하고, 모든 출력은 웰니스 가드레일(SAD-010)을 통과한 뒤 사용자에게 노출된다. LLM 사용 시 최신 Claude 모델을 기본으로 한다.

## 컴포넌트 / 책임
- **Program Generator**: 목표/경력/장비/일수 → 점진적 과부하 기반 요일별 종목·세트·반복·강도 생성. 장비 제약·부상 제외 시 대체 운동 재구성(SRS-009).
- **Periodization Engine**: 직전 주 완수율·RPE·달성 무게로 다음 주 강도/볼륨 자동 조정, 주기적 디로드 편성, 정체 감지 시 볼륨 감소/회복 권고(SRS-010).
- **Exercise Selector**: 운동 카탈로그(SAD-002)·대체운동 그래프를 사용해 장비/근육군 균형 충족.
- **Guardrail Filter(연계)**: 생성 텍스트의 진단·치료·보장 표현 차단(SAD-010 위임).

## 주요 결정 포인트 (→ ADR 후보)
- 프로그램 생성 방식: 규칙엔진/템플릿 우선 + LLM 보조 하이브리드 채택(설명가능성·안전) → ADR.
- 주기화·과부하 진행 규칙의 파라미터화·근거 출처 → ADR.

## 연결
- 다루는 SRS: SRS-009(AI 프로그램 생성), SRS-010(점진과부하·주기화)
