# SAD-009 · 수익화 & 구독/엔타이틀먼트

## 개요 (Summary)
수익화 계층은 **무료 티어로 진입장벽을 0으로** 유지해 소셜·SEO 네트워크 효과를 키우고, **프리미엄 구독(AI 코치 심화·고급 분석)**과 **그룹/챌린지 유료·PT 마켓**으로 수익을 만든다(SRS-014). 결제는 앱스토어/플레이 인앱결제로 처리하고, 서버는 영수증 검증 후 **엔타이틀먼트(권한)**를 발급해 기능 잠금을 제어한다. 전환 전략은 하드페이월보다 "그룹 가치로 묶는 리텐션" 우선(reference 교훈).

## 컴포넌트 / 책임
- **Entitlement Service**: 사용자별 구독 상태·기능 권한 판정. AI 코치 심화·고급 분석 잠금 해제(SRS-014).
- **IAP Integration**: 앱스토어/플레이 인앱결제 연동, 영수증 검증, 해지·복원 처리.
- **Paywall & Tiering**: 무료/프리미엄 경계 정의, 무료 티어로 코어 로깅·기본 분석·기본 소셜 제공.
- **Group/PT Monetization**: 유료 그룹·챌린지·PT 연결 등 수익 경로 처리(SAD-005 연계).

## 주요 결정 포인트 (→ ADR 후보)
- 무료/프리미엄 기능 경계와 전환 전략(그룹 가치 우선 vs 하드페이월) → ADR.
- 구독 인프라 자체 구축 vs 서드파티(RevenueCat 등) → ADR.
