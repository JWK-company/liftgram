# 착용장비 브랜드 자동 식별 정확도 — 블라인드 실측

- 일자: 2026-07-20
- 목적: **ADR-027 D5**("브랜드·모델을 특정하지 않고 카테고리 수준만 태그한다")의 개정 여부 판단
- 채점자: Claude Opus 4.8 (비전)
- 표본: 14장 · 블라인드(파일명 무힌트, 정답지 미열람 상태로 예측 고정 후 채점)
- 원자료: 세션 스크래치패드 `geartest/` — `MY_PREDICTIONS.json`(예측 고정본) · `ANSWER_KEY.json` · `SCORING_RULES.md`(사전 확정 기준) · `COLLECTION_NOTES.md`

## 왜 이 측정을 했나

D5의 근거는 두 가지였다.
1. API 없이 수동 큐레이션만으로는 브랜드·모델 단위 상품 매핑의 유지 비용이 감당되지 않는다.
2. **사진 속 로고를 특정 브랜드로 단정하면 오인식 시 상표·표시광고 분쟁 소지가 생긴다.**

근거 1은 딥링크에만 해당한다 — 검색 URL은 브랜드명을 쿼리에 넣기만 하면 되므로 유지 비용이 0이다.
근거 2가 실질 근거이며, 이는 **"얼마나 잘 맞히나"가 아니라 "얼마나 틀리게 단정하나"** 로 검증해야 한다.
따라서 핵심 지표를 **브랜드 오단정률**로 두고, 판정선을 결과 확인 **전에** 확정했다(≤10% → 개정).

## 이 측정의 한계 (결과 해석 시 반드시 함께 읽을 것)

1. **낙관 편향 — 가장 중요.** 14장 중 12장이 브랜드 공식몰 커머셜 사진이다. 착용컷 8장조차 로고가 카메라를 향하도록 설계된 광고 촬영물이라, 실사용자 사진의 특징(모션 블러·역광·거울 셀카·마모된 로고)이 거의 없다. **이 수치는 상한선으로 읽어야 한다.**
2. **모델 등급 차이.** 채점자는 Opus 4.8이다. 프로덕션은 비용 때문에 더 작은 모델을 쓸 가능성이 높고, 그 경우 수치는 하락한다. **배포 전 대상 모델로 재측정이 필수다.**
3. **표본 14장**은 셀별 정확도를 결론으로 삼기에 부족하다. 방향성 판단용이다.
4. **국내 브랜드 0종.** 한국 서비스의 실사용 분포를 대표하지 못한다. 국내 브랜드·저가 노브랜드 대역은 별도 측정이 필요하다(Cafe24 기반 쇼핑몰은 브랜드를 기계적으로 확정할 표준 메타데이터가 없어 이번 수집에서 배제됨).
5. 정답은 **브랜드 자사몰의 자체 제품 페이지**(Shopify `products.json`의 `vendor`)에서만 확보했다. 추측 정답 0건.
