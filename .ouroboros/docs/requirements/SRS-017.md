# SRS-017 · 다이렉트 메시지 (1:1·그룹 채팅·이미지·PT문의/만남 조율)

## 요구 서술
시스템은 사용자 간 **1:1 및 그룹 다이렉트 메시지**(텍스트·이미지)를 제공하고, 메시지를 실시간에 가깝게 전달·동기화하며 전송/읽음 상태를 표시한다. 사용자는 대화에서 **PT 문의·만남 약속(일정 제안)** 을 주고받을 수 있고, 새 메시지는 통합 알림(SRS-020)으로 통지된다. 스팸·안전 통제를 위해 **수신 범위(팔로워/전체) 설정·요청 메시지(승인 전 제한)·신고·차단**을 제공하며, 차단 시 대화가 중단된다. 음성/영상 통화는 본 요구 범위에서 제외한다(후속). PT 거래로 이어질 경우 결제·정산은 트레이너 수익화(SRS-016)로 인계된다.

## 수용 기준
- 사용자는 1:1·그룹 대화방에서 텍스트·이미지 메시지를 주고받고 전송/읽음 상태를 확인한다.
- 사용자는 메시지에서 PT 문의 및 만남(일정 제안)을 교환한다.
- 사용자는 메시지 수신 범위를 설정하며, 미허용 발신자의 메시지는 요청 메시지로 제한된다.
- 사용자는 메시지/상대를 신고·차단할 수 있고, 차단 시 대화가 중단된다.
- 새 메시지·답장은 통합 알림(SRS-020)으로 전달된다.
