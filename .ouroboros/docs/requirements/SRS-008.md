# SRS-008 · 프로필 및 PR 공유

## 요구 서술
시스템은 사용자별 **프로필**(누적 통계·주요 PR 보드·최근 운동)을 제공하고, 사용자가 자신의 PR/진척 카드를 소셜로 **공유**할 수 있게 한다. 프로필 및 각 기록의 **공개 범위(전체/팔로워/비공개)**를 사용자가 제어한다. PR 공유는 분석(SRS-005)에서 생성된 사실 기반 수치를 사용한다.

## 수용 기준
- 프로필에 누적 운동 수·주요 종목 PR·최근 활동이 표시된다.
- 사용자는 프로필/기록의 공개 범위를 전체/팔로워/비공개로 설정할 수 있다.
- PR/진척 카드를 1탭으로 피드에 공유할 수 있다.

## 연결
- 도출 URS: URS-003(함께 운동), URS-002(진척/동기부여)
