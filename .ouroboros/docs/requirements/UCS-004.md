# UCS-004 · AI 프로그램을 생성하고 주차별로 진행한다

## 액터
무엇을 할지 정해주길 원하는 사용자(P2). 부차 액터: AI 프로그래밍 엔진.

## 사전 조건
- 로그인되어 있고 웰니스 면책·안전 고지에 동의했다.

## 주 흐름
1. 사용자가 "AI 코치로 프로그램 만들기"를 선택한다.
2. 목표(근비대/근력/체중관리)·운동 경력·가용 장비·주당 가능 일수를 입력한다.
3. 엔진이 점진적 과부하 원칙에 기반한 다주차 프로그램(요일별 종목·세트·반복·강도)을 생성한다.
4. 사용자가 프로그램을 검토하고 일부 종목을 교체/조정한 뒤 채택한다.
5. 채택된 프로그램의 오늘 세션이 루틴으로 제공되어 UCS-001 흐름으로 수행한다.
6. 주차가 끝나면 직전 주 수행 데이터(완수율·RPE)를 바탕으로 다음 주 강도/볼륨이 자동 조정되고, 주기에 맞춰 디로드 주차가 편성된다.

## 대안 / 예외 흐름
- 3a. 입력이 부족하면 합리적 기본값을 제안하고 사용자가 확인한다.
- 4a. 사용자가 특정 종목을 부상 등으로 제외하면 대체 종목으로 재구성한다.
- 6a. 완수율이 지속적으로 낮으면 볼륨을 낮추고 회복 권고를 제시한다(진단 표현 없이, 웰니스 범위).

## 사후 조건
- 활성 프로그램이 사용자에 연결되고 주차 진행 상태가 추적된다.
- 모든 권고 문구가 웰니스 가드레일(SRS-015)을 통과한다.

## 연결
- 구체화 대상 URS: URS-004(코치/프로그램)
