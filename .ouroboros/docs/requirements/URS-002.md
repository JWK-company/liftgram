# URS-002 · 내 성장을 데이터로 확인하고 동기부여 받고 싶다

## 배경 / 이해관계자
사람들은 "기록 자체"가 아니라 "내가 늘고 있다는 증거" 때문에 운동을 지속한다. 흩어진 기록만으로는 볼륨·중량 추세나 개인 최고기록(PR) 갱신을 체감하기 어렵다. 진척이 보이지 않으면 동기가 꺼지고 이탈한다(헬스앱 공통 약점, reference/hevy-success-analysis §7). 이해관계자는 자신의 데이터가 의미 있는 그래프·지표로 환원되기를 원한다.

## 요구 서술
사용자는 자신의 운동 기록이 자동으로 **볼륨·추정 1RM·PR·부위별 분포·기간 추세**로 집계되어, 별도 계산 없이 진척을 한눈에 보고 PR 갱신 순간 즉시 인지할 수 있어야 한다. 지표는 신뢰할 수 있는 산식에 기반해야 하며, 과장 없는 사실적 수치로 제시되어야 한다(웰니스 포지션).

## 수용 기준
- 종목별 추정 1RM과 PR(최대중량·최대볼륨)이 기록 저장 즉시 갱신되어 표시된다.
- 주/월 단위 총 볼륨·세션 수·부위별 분포 추세 그래프를 제공한다.
- PR 갱신이 발생하면 세션 종료 화면에서 사용자에게 명시적으로 알린다.
