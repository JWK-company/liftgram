# SRS-004 · 운동 세션 관리 (진행중 저장·복구·요약)

## 요구 서술
시스템은 운동 세션의 **시작·진행·종료** 생애주기를 관리한다. 루틴 기반 시작과 즉흥(빈 운동) 시작을 모두 지원하고, 세션 진행 상태를 지속적으로 보존하여 앱 종료·기기 변경 후에도 **손실 없이 복구**한다. 세션 종료 시 총 볼륨·소요 시간·PR 갱신 여부를 담은 **요약**을 제공한다.

## 수용 기준
- 진행 중 세션은 앱 강제 종료/재실행 후에도 입력값 손실 없이 복구된다.
- 루틴 없이 빈 운동을 시작하고 세션 중 종목을 추가/삭제할 수 있다.
- 세션 종료 시 총 볼륨·소요 시간·PR 갱신 여부 요약이 표시된다.

## 연결
- 도출 URS: URS-001(빠른 로깅)
