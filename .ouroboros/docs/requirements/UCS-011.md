# UCS-011 · 미디어·스토리를 게시하고 탐색/검색으로 발견한다

## 액터
콘텐츠를 올리고 사람·콘텐츠를 발견하려는 사용자(P3). 부차 액터: 팔로워·탐색 사용자.

## 사전 조건
- 로그인되어 있고 프로필이 생성되어 있다.
- 게시 공개범위(전체/팔로워/비공개)가 설정되어 있다.

## 주 흐름
1. 사용자가 사진/영상 게시물 또는 24시간 스토리를 올린다(운동 기록 외 콘텐츠 포함). 캡션·해시태그·멘션을 단다.
2. 공개범위에 따라 팔로워의 피드·스토리 트레이에 노출된다.
3. 다른 사용자가 탐색(추천)·검색(사용자/해시태그/콘텐츠)으로 콘텐츠와 사람을 발견한다.
4. 발견한 사용자를 팔로우하고 프로필 → DM(UCS-010) 동선으로 연결된다.

## 대안 / 예외 흐름
- 1a. 공개범위가 비공개면 승인된 팔로워에게만 노출된다.
- 2a. 모더레이션 위반 콘텐츠(ADR-017)는 게시 차단되거나 사후 내림 처리된다.
- 3a. 탐색은 부족/팔로우 맥락을 우선하고 넓은 글로벌 추천은 보조로 둔다("넓은 피드 공허" 방지 — ADR-014).

## 사후 조건
- 게시물은 저장·노출되고, 스토리는 24시간 후 자동 만료(비노출)된다.
- 발견·노출 지표(조회·도달)가 갱신된다.

## 연결
- 구체화 대상 URS: URS-010(직접 연결·발견), URS-003(함께 운동)
