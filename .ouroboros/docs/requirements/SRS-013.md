# SRS-013 · 한국 로컬라이즈 (한국어DB·기구기반루틴·오운완)

## 요구 서술
시스템은 **한국어 UI와 한국어 운동명/별칭 DB**를 제공하고, 사용자가 지정한 **헬스장 가용 기구**에 맞춰 수행 가능한 루틴/대체 운동만 추천한다. 한국 헬스 커뮤니티 문화인 **오운완 인증**을 1탭으로 게시해 소셜 루프(SRS-007/011)에 노출한다. 날짜·무게 단위 등 표기를 한국 사용자 관습에 맞춘다.

## 수용 기준
- UI 전체와 운동 DB가 한국어를 지원하고 한국어 운동명으로 검색된다.
- 사용자가 가용 기구를 지정하면 그 기구로 가능한 루틴/대체 운동만 추천된다.
- 오운완 인증을 1탭으로 게시하여 피드/그룹에 노출할 수 있다.

## 연결
- 도출 URS: URS-007(한국 환경)
