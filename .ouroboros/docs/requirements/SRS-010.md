# SRS-010 · 점진적 과부하·자동 주차 진행·디로드

## 요구 서술
시스템은 활성 프로그램의 주차가 진행될 때 **직전 주 수행 데이터(완수율·RPE·달성 무게)**를 기반으로 다음 주의 강도/볼륨을 **자동 조정**하고, 설정된 주기마다 **디로드 주차**를 편성한다. 완수율이 지속적으로 낮거나 정체가 감지되면 볼륨을 낮추거나 회복을 권고한다(웰니스 범위, 진단 표현 없이). 진행 규칙은 사용자에게 투명하게 설명된다.

## 수용 기준
- 직전 주 완수율·RPE에 따라 다음 주 강도/볼륨이 규칙대로 자동 조정된다.
- 설정 주기(예: N주)마다 디로드 주차가 자동 편성된다.
- 연속 미달성/정체 시 볼륨 감소 또는 회복 권고가 제시되며 진단·치료 표현을 쓰지 않는다.

## 연결
- 도출 URS: URS-004(코치/프로그램), URS-002(진척/동기부여)
