# SRS-007 · 소셜 피드 (팔로우·피드·좋아요/댓글·공유)

## 요구 서술
시스템은 사용자가 다른 사용자를 **팔로우**하고 팔로잉의 완료 운동·PR·인증을 **시간순 피드**로 보며, 항목에 **좋아요·댓글**로 반응하고 작성자에게 **알림**이 전달되게 한다. 사용자는 자신의 운동/PR을 1탭으로 피드에 공유한다. 팔로잉이 적어 피드가 빌 경우 같은 부족/그룹 기반 추천으로 채워 "넓은 피드 공허"를 방지한다. 부적절 콘텐츠는 신고·차단으로 관리한다.

## 수용 기준
- 사용자는 팔로우/언팔로우하고 팔로잉의 완료 운동을 시간순 피드에서 본다.
- 좋아요·댓글 작성 시 대상 작성자에게 알림이 전달된다.
- 운동/PR을 1탭으로 공유하고, 피드 항목을 신고·차단할 수 있다.
- 팔로잉 부족 시 부족/그룹 기반 추천이 피드를 보완한다.

## 연결
- 도출 URS: URS-003(함께 운동)
