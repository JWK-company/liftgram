# UCS-002 · 진척 분석을 조회하고 PR을 확인한다

## 액터
자신의 성장을 확인하려는 사용자(P1·P2).

## 사전 조건
- 사용자에게 최소 1건 이상의 완료 세션 기록이 있다.

## 주 흐름
1. 사용자가 "분석/대시보드"를 연다.
2. 기간(주/월/전체)을 선택한다.
3. 총 볼륨·세션 수·부위별 분포·기간 추세 그래프가 표시된다.
4. 특정 종목을 선택하면 추정 1RM 추세와 PR(최대중량·최대볼륨·최대반복)이 표시된다.
5. 사용자가 PR 항목을 탭하면 해당 PR이 달성된 세션으로 이동한다.

## 대안 / 예외 흐름
- 1a. 세션을 막 종료한 경우, 세션 요약 화면에서 이번 PR 갱신이 즉시 강조된다(별도 진입 불필요).
- 3a. 데이터가 부족(기록 1~2건)하면 "더 많은 기록이 쌓이면 추세가 보입니다" 안내를 표시한다.

## 사후 조건
- 조회만 발생하며 기록은 변경되지 않는다.
- (선택) 사용자가 PR/분석 카드를 소셜로 공유할 수 있다(UCS-003 연계).

## 연결
- 구체화 대상 URS: URS-002(진척/동기부여)
