# UCS-010 · DM으로 PT 문의·만남 약속을 잡고 동료와 소통한다 (1:1·그룹)

## 액터
직접 연결을 원하는 사용자(P3·PT 수요자). 부차 액터: 트레이너(공급 측, URS-009)·동료·그룹 멤버.

## 사전 조건
- 로그인되어 있고 프로필이 생성되어 있다.
- 상대방이 메시지 수신을 허용하는 범위(팔로워/전체) 안에 있거나, 요청 메시지 경로가 열려 있다.

## 주 흐름
1. 사용자가 피드·탐색·검색·프로필에서 동료 또는 트레이너를 발견한다.
2. 프로필에서 'DM' 또는 'PT 문의' 버튼으로 1:1 대화방에 진입한다.
3. 텍스트·이미지로 PT 문의·운동 상담을 하고, 만남/세션 일정(제안)을 주고받는다.
4. 필요 시 그룹 채팅을 만들어 여러 명과 만남·운동 약속을 조율한다.
5. 새 메시지·답장은 통합 알림(SRS-020)으로 통지되고, 전송/읽음 상태가 표시된다.

## 대안 / 예외 흐름
- 2a. 상대가 수신을 '팔로워'로 제한했고 사용자가 미팔로우면, 승인 전 제한된 **요청 메시지**로 전송된다.
- 3a. 스팸·부적절 메시지는 신고·차단으로 처리되며, 차단 시 대화가 중단된다.
- 3b. 미성년·민감정보 보호 정책(ADR-017)에 따라 특정 연결/콘텐츠 공유가 제한될 수 있다.
- 3c. PT 거래로 이어지면 결제·정산은 트레이너 수익화(SRS-016) 흐름으로 인계된다.

## 사후 조건
- 대화·일정 약속이 기록되고 알림으로 전파된다.
- 차단/신고 시 안전 상태(차단 목록·신고 큐)가 반영된다.

## 연결
- 구체화 대상 URS: URS-010(직접 연결·DM·PT문의·만남), URS-003(함께 운동)
