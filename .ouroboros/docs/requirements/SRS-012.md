# SRS-012 · 자동 횟수 카운팅 (센서/카메라·수동폴백)

## 요구 서술
시스템은 **지원 종목에 한해** 기기 센서/카메라로 반복 동작을 인식하여 **반복 횟수를 자동 카운팅**하고 세트 기록에 채운다. 자동값은 저장 전 사용자가 1탭으로 보정할 수 있고, 미지원 종목·권한 거부·인식 실패 시 **수동 입력으로 즉시 폴백**하여 기록 흐름을 막지 않는다. 본 기능은 횟수 카운팅에 한정하며 **자세 평가·부상 진단을 표방하지 않는다**(웰니스 가드레일 SRS-015 준수). 지원 종목은 정확도가 검증된 소수에서 점진 확대한다.

## 수용 기준
- 지원 종목에서 세트 수행 시 반복 횟수가 자동 집계되어 입력란에 채워진다.
- 자동 집계값을 저장 전 1탭으로 가감/수정할 수 있다.
- 미지원/권한거부/인식실패 시 수동 입력으로 자동 폴백된다.
- 기능 설명·결과에 자세 진단·부상 판정 표현이 없다.

## 연결
- 도출 URS: URS-006(로깅 마찰 감소)
