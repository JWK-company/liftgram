# UCS-001 · 루틴을 만들고 운동 세션을 수행한다

## 액터
운동 기록 사용자(P1). 부차 액터: 직전 기록 데이터, (선택) 자동 카운팅 모듈.

## 사전 조건
- 로그인되어 있고 운동 DB에 접근 가능하다.
- 최소 1개 이상의 종목이 운동 DB에 존재한다.

## 주 흐름
1. 사용자가 "새 루틴"을 만들고 운동 DB에서 종목을 검색·추가한다.
2. 각 종목에 목표 세트 수/반복 범위를 지정하고 순서를 정렬한다(슈퍼셋 묶기 포함).
3. 루틴을 저장한다.
4. 사용자가 저장된 루틴으로 "운동 시작"을 누른다.
5. 각 종목 입력란 옆에 직전 세션 동일 종목 기록이 자동 표시된다.
6. 사용자가 세트별 무게·횟수·RPE를 입력하고 세트 완료를 체크하면 휴식 타이머가 시작된다.
7. 모든 종목 완료 후 세션을 종료하면 기록이 저장되고 요약(총 볼륨·소요시간·PR 여부)이 표시된다.

## 대안 / 예외 흐름
- 2a. 목표 기구가 없으면 대체 운동 토글로 즉석 교체한다.
- 4a. 루틴 없이 "빈 운동 시작"으로 즉흥 세션을 진행하고 종목을 그때그때 추가한다.
- 6a. (URS-006 연계) 지원 종목이면 자동 카운팅으로 횟수가 채워지고 사용자는 보정만 한다.
- 6b. 앱이 종료/중단되면 다음 실행 시 진행 중 세션이 복구된다.

## 사후 조건
- 완료 세션이 사용자 기록에 영구 저장되고 분석 지표(볼륨·1RM·PR)에 반영된다.
- 이번 세션 값이 다음 세션의 "직전 기록"이 된다.

## 연결
- 구체화 대상 URS: URS-001(빠른 로깅), URS-006(로깅 마찰 감소)
