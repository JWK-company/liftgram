# SRS-009 · AI 프로그램 생성 (목표·경력·장비·일정 → 프로그램)

## 요구 서술
시스템은 사용자의 **목표(근비대/근력/체중관리 등)·운동 경력·가용 장비·주당 가능 일수**를 입력받아, 점진적 과부하 원칙에 기반한 **다주차 구조화 프로그램**(요일별 종목·세트·반복·강도)을 자동 생성한다. 생성된 프로그램은 사용자가 검토·종목 교체·조정 후 채택할 수 있으며, 부상 제외 종목은 대체 종목으로 재구성된다. 모든 생성 결과와 문구는 웰니스 가드레일(SRS-015)을 통과해야 한다.

## 수용 기준
- 목표·경력·장비·일수 입력 후 4주 이상 요일별 프로그램(종목·세트·반복·강도)이 생성된다.
- 사용자는 생성된 프로그램의 종목을 대체 운동으로 교체/제외할 수 있고 즉시 재구성된다.
- 생성 결과에 진단·치료·질병 개선 표현이 포함되지 않는다(가드레일 통과).

## 연결
