# SRS-003 · 세트 로깅 (무게/횟수/RPE·직전기록·휴식타이머)

## 요구 서술
시스템은 운동 세션 중 종목별로 **세트 단위 무게·반복 횟수·(선택)RPE**를 최소 조작으로 입력·저장하게 한다. 각 종목 입력란에는 **직전 동일 종목 세션의 기록이 자동 표시**되어 별도 조회 없이 참고·자동 채움 기준으로 쓰인다. 세트 완료 시 **휴식 타이머**가 자동 시작되고, 무게 단위(kg/lb)와 바벨 플레이트 계산을 지원한다.

## 수용 기준
- 직전값 자동 채움 기준 세트 1건 저장에 필요한 조작이 ≤ 3탭이다.
- 각 종목에 직전 세션 기록(세트별 무게×횟수)이 입력란 옆에 자동 표시된다.
- 세트 완료 시 설정된 휴식 타이머가 자동 시작되고 종료를 알린다.
- kg/lb 단위 전환과 워밍업/실패 세트 표기를 지원한다.

## 연결
- 도출 URS: URS-001(빠른 로깅), URS-006(로깅 마찰 감소)
