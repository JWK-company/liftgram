# UCS-006 · 자동 횟수 카운팅으로 세트를 기록한다

## 액터
손 입력 없이 기록하려는 사용자. 부차 액터: 기기 센서/카메라, 카운팅 모듈.

## 사전 조건
- 수행 종목이 자동 카운팅 지원 목록에 있다.
- 카메라/센서 권한이 허용되어 있다(미허용 시 수동 폴백).

## 주 흐름
1. 사용자가 세트 시작 시 "자동 카운팅"을 켠다.
2. 모듈이 센서/카메라로 반복 동작을 인식해 실시간 카운트를 표시한다.
3. 세트 종료 시 인식된 반복 횟수가 입력란에 자동으로 채워진다.
4. 사용자가 값(필요 시 무게 포함)을 확인하고 1탭으로 저장한다.

## 대안 / 예외 흐름
- 1a. 미지원 종목/권한 거부 시 자동으로 수동 입력 모드로 폴백한다(흐름 차단 없음).
- 2a. 인식 신뢰도가 낮으면 경고 표시 후 사용자가 수동 보정한다.
- 3a. 명백한 오인식(과다/과소 카운트)은 사용자가 +/− 로 1탭 보정한다.

## 사후 조건
- 자동/보정된 횟수가 일반 세트 기록과 동일하게 저장되어 분석에 반영된다.
- 자세 평가·부상 진단은 수행/표방하지 않는다(웰니스 가드레일 SRS-015 준수).

## 연결
