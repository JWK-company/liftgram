# SRS-006 · 계정·인증 및 로컬우선 크로스디바이스 싱크

## 요구 서술
시스템은 이메일/소셜 기반 **가입·로그인**을 제공하고, 운동 데이터를 **로컬 우선(offline-first)**으로 저장하여 네트워크 없이도 기록을 이어가게 한다. 온라인 복귀 시 데이터를 서버와 **양방향 동기화**하여 여러 기기에서 끊김 없이 같은 기록·진척을 본다. 동시 편집 충돌은 결정적 규칙으로 해소한다.

## 수용 기준
- 오프라인 상태에서 세션 로깅이 가능하고 온라인 복귀 시 자동 동기화된다.
- 두 기기에서 로그인 시 동일한 기록·루틴·분석이 일관되게 표시된다.
- 동기화 충돌이 데이터 손실 없이 결정적 규칙으로 병합/해소된다.

## 연결
- 도출 URS: URS-001(빠른 로깅), URS-002(진척/동기부여)
