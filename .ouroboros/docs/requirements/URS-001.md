# URS-001 · 운동을 빠르고 간편하게 기록하고 싶다

## 배경 / 이해관계자
운동 기록 습관이 있는 중급 리프터(P1)는 헬스장에서 세트 사이 짧은 휴식 중에 무게·횟수를 기록한다. 기존 종이·엑셀·메모앱은 입력이 느리고 직전 기록을 다시 찾아봐야 해서 흐름이 끊긴다. Hevy·Strong이 이 시장을 검증했으나(상품화됨), 우리도 최소한 그 수준의 마찰 없는 로깅을 제공하지 못하면 진입 자체가 불가능하다. 이해관계자는 "기록은 거추장스럽지 않아야 한다"고 요구한다.

## 요구 서술
사용자는 운동 중 세트 단위로 무게·횟수·RPE를 **최소 탭으로 즉시** 기록하고, 직전 운동의 같은 종목 기록을 별도 조회 없이 **바로 옆에서 참고**할 수 있어야 한다. 기록은 어떤 기기에서 시작해도 끊김 없이 이어져야 한다. (이것은 니즈이며, 구체적 UI/저장방식은 SRS·SAD에서 정한다.)

## 수용 기준
- 세트 1건 기록에 필요한 사용자 조작이 ≤ 3탭(직전값 자동 채움 기준).
- 운동 시작 시 각 종목의 직전 세션 기록이 입력란 옆에 자동 표시된다.
- 앱을 닫았다 다시 열어도 진행 중이던 세션이 손실 없이 복구된다.
