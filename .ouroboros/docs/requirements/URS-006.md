# URS-006 · 기록 자체의 마찰(수동 입력)을 줄이고 싶다

## 배경 / 이해관계자
모든 경쟁 앱(Hevy 포함)은 세트마다 무게·횟수를 수동 탭으로 입력한다. 운동 중 땀과 피로 속에서 매 세트 숫자를 치는 것은 흐름을 끊고 누락·오기를 부른다. 사용자 메모에서도 "카운팅 기능"이 차별점으로 제시되었다(reference/health-app §사용자 메모). 이해관계자는 횟수를 손으로 세지 않고도 자동으로 기록되기를 원한다. 단 정확도·부상 책임 리스크가 있어 좁게 시작해야 한다.

## 요구 서술
사용자는 지원되는 종목에 한해 센서/카메라 기반으로 **반복 횟수가 자동 카운팅되어 세트 기록에 채워지는** 경험을 선택적으로 사용할 수 있어야 한다. 자동 카운팅은 항상 수동 보정이 가능해야 하며, 인식 실패 시에도 기록 흐름이 막히지 않아야 한다. 자세 평가·부상 진단을 표방하지 않는다(웰니스 포지션).

## 수용 기준
- 지원 종목에서 1세트 수행 시 반복 횟수가 자동 집계되어 입력란에 채워진다.
- 자동 집계값은 저장 전 사용자가 1탭으로 수정할 수 있다.
- 자동 카운팅 미지원/실패 시 수동 입력으로 즉시 폴백된다.
