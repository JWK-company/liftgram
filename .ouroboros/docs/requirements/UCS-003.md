# UCS-003 · 동료를 팔로우하고 피드에서 소통한다

## 액터
소셜로 자극받는 사용자(P3). 부차 액터: 팔로우 대상 사용자.

## 사전 조건
- 로그인되어 있고 프로필이 생성되어 있다.

## 주 흐름
1. 사용자가 동료를 검색하거나 추천 목록에서 찾아 팔로우한다.
2. 홈 피드에 팔로잉의 완료 운동·PR·오운완 인증이 시간순으로 나타난다.
3. 사용자가 피드 항목에 좋아요/댓글을 단다.
4. 작성자에게 알림이 전달된다.
5. 사용자가 자신의 운동/PR을 1탭으로 피드에 공유한다.

## 대안 / 예외 흐름
- 1a. 팔로우 대상이 비공개 계정이면 팔로우 요청 후 승인 시 피드에 노출된다.
- 2a. 팔로잉이 적어 피드가 비면, 같은 부족/그룹 기반 추천으로 채운다(넓은 피드 공허 방지).
- 3a. 부적절 댓글은 신고/차단으로 처리된다.

## 사후 조건
- 상호작용(좋아요·댓글·공유)이 기록되고 알림으로 전파된다.
- 소셜 그래프(팔로우 관계)가 갱신된다.

## 연결
- 구체화 대상 URS: URS-003(함께 운동)
