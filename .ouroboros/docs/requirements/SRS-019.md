# SRS-019 · 미디어 게시물·스토리 (사진/영상 포스트·24h 스토리)

## 요구 서술
시스템은 사용자가 **사진·영상 게시물**과 **24시간 후 만료되는 스토리**를 올리도록 한다(운동 기록 외 일상·몸 사진·짧은 영상 포함). 게시물은 캡션·해시태그·멘션·공개범위(전체/팔로워/비공개)를 가지며, 미디어는 업로드·저장·전송(썸네일/트랜스코딩) 파이프라인을 거친다(SAD-012). 게시물·스토리는 피드(SRS-007)·스토리 트레이·프로필(SRS-008)에 노출되고, 모든 콘텐츠는 모더레이션·안전·개인정보 정책(ADR-017)을 통과해야 한다(신고·내림 가능).

## 수용 기준
- 사용자는 사진/영상 게시물을 캡션·해시태그·멘션·공개범위와 함께 올린다.
- 사용자는 24시간 스토리를 올리고, 스토리는 만료 후 자동 비노출된다.
- 업로드 미디어가 적절히 저장·표시(썸네일/재생)되며, 콘텐츠를 신고·내림할 수 있다.

## 연결
- 도출 URS: URS-010(직접 연결·발견), URS-003(함께 운동)
- 관련: SRS-007(피드 노출) · SRS-008(프로필) · SAD-012(미디어 파이프라인) · ADR-016(미디어/CDN) · ADR-017(모더레이션)
