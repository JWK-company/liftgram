# SRS-005 · 분석 대시보드 (볼륨·추정1RM·PR·추세)

## 요구 서술
시스템은 사용자의 완료 기록을 자동 집계하여 **총 볼륨·추정 1RM·개인 최고기록(PR: 최대중량/최대볼륨/최대반복)·부위별 분포·기간 추세**를 시각화한다. 추정 1RM은 명시된 표준 산식(예: Epley)으로 계산하며, 종목을 선택하면 해당 종목의 1RM 추세와 PR 이력으로 드릴다운한다. 모든 수치는 과장 없이 사실 기반으로 표시한다(웰니스).

## 수용 기준
- 종목별 추정 1RM·PR이 기록 저장 즉시 갱신되어 표시된다.
- 주/월/전체 기간의 총 볼륨·세션 수·부위별 분포 추세 그래프를 제공한다.
- 1RM 추정 산식이 문서화되고 동일 입력에 대해 일관된 값을 산출한다.

## 연결
- 도출 URS: URS-002(진척/동기부여)
