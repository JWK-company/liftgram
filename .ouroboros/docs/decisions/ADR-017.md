# ADR-017 · 콘텐츠 모더레이션·안전·민감정보 정책 (자동필터+신고큐·DM 안전·미성년/건강 보호)

## 맥락 (Context)
옵트인 공개 레이어(ADR-014)·미디어(SRS-019)·DM(SRS-017)·피드(SRS-007)는 스팸·부적절 콘텐츠·괴롭힘·미성년·개인정보(건강=민감정보) 리스크를 키운다. 한국 규제(디지털의료제품법·의료기기법·개인정보보호 — reference/health-app·risks)와 웰니스 가드레일(ADR-006)을 위반하면 법적·신뢰 비용이 크다. 신고·차단만으로는 규모에서 부족하다.

## 결정 (Decision)
**다층 모더레이션·안전·개인정보 정책**을 채택한다.
- ① **콘텐츠 모더레이션**: 업로드/게시 시 자동 필터(해시·분류) + 신고 큐·사람 검수·내림/서스펜드(SRS-019).
- ② **DM 안전**: 수신범위(팔로워/전체)·요청 메시지·신고·차단으로 스팸/괴롭힘 차단(SRS-017).
- ③ **민감정보·미성년**: 건강=민감정보 동의·최소수집·공개범위 기본 보수화, 미성년 보호(연령 게이트·DM/탐색 제한).
- ④ **톤·표현**: 모든 카피·케어/넛지 알림(SRS-020)은 웰니스 가드레일(ADR-006)·지지형 톤 준수(진단·압박·수치심 표현 금지).

## 근거 / 대안
- 채택 이유: 규제·신뢰·안전은 공개 소셜의 전제. 자동+사람 혼합이 규모와 정확도(맥락)의 균형을 준다.
- 검토한 대안: ① **신고 기반 사후 대응만** — 규모·법적 리스크 미흡. ② **전면 자동 모더레이션** — 오탐·맥락 부족. ③ **공개 레이어 포기** — ADR-014와 충돌.

## 결과 (Consequences)
- 장점: 법적·신뢰 리스크 완화, 안전한 공개 레이어 운영.
- 트레이드오프: 모더레이션 운영비·검수 인력·오탐 마찰.
- 되돌림 비용: 낮음 — 정책 강화는 점진 가능.

## 연결
- 영향 SRS/SAD: SAD-011·SAD-012, SRS-019·SRS-020·SRS-007
- 정합: ADR-006(웰니스 가드레일)·ADR-014(공개 레이어) · 대체: 없음
