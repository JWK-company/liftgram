# ADR-005 · 자동 카운팅은 온디바이스 추론 + 초기 지원 종목 협소화

## 맥락 (Context)
자동 횟수 카운팅(SRS-012)은 운동 중 신체 영상을 다룬다 — 민감정보·프라이버시·지연·오프라인 이슈가 크다. 또한 정확도가 낮으면 신뢰를 잃고, 종목별 인식 난이도·부상 책임 리스크가 다르다(reference: 중상 난이도·부상 책임). 좁게 시작하라는 권고가 있다.

## 결정 (Decision)
카운팅 추론을 **온디바이스**로 수행하고 **원시 영상은 저장·전송하지 않는다**. 지원 종목은 **정확도가 검증된 소수에서 시작해 점진 확대**한다. 미지원·실패·권한 거부 시 수동 입력으로 무중단 폴백한다.

## 근거 / 대안
- 채택 이유: 온디바이스는 프라이버시·지연·오프라인에서 우월하고 웰니스/민감정보 정책(ADR-006)과 정합. 협소 시작은 정확도·부상 리스크를 통제.
- 검토한 대안: ① **서버 추론** — 모델 관리 쉬우나 영상 전송·프라이버시·지연·비용 부담. ② **광범위 종목 즉시 지원** — 마케팅 매력적이나 오인식·부상·신뢰 붕괴 위험.

## 결과 (Consequences)
- 장점: 프라이버시 강점, 오프라인 동작, 리스크 통제된 점진 확대.
- 트레이드오프: 온디바이스 모델 최적화·기기 성능 편차 대응, 초기 커버리지 제한.
- 되돌림 비용: 중 — 추론 위치는 캡슐화 시 조정 가능하나 프라이버시 약속 변경은 신뢰 비용.

## 연결
- 영향 SRS/SAD: SAD-007(자동 카운팅), SRS-012 · 대체: 없음
