# ADR-010 · 추정 1RM 산식 표준 = Epley (표시·일관성 정책)

## 맥락 (Context)
분석(SRS-005)의 추정 1RM은 진척 지표·PR 판정·AI 프로그래밍 강도 산정의 기준이 된다. 1RM 추정 공식은 여러 가지(Epley, Brzycki, Lombardi 등)가 있고 결과가 미세히 다르다. 동일 입력에 다른 값이 나오면 신뢰가 무너지고, 과장된 수치는 웰니스(ADR-006) 위반 소지가 있다.

## 결정 (Decision)
추정 1RM 표준 산식을 **Epley(1RM = w × (1 + reps/30))**로 고정하고, UI에 "추정치"임을 명시한다. 모든 모듈(분석·PR·AI 강도)은 동일 산식을 단일 출처로 공유한다.

## 근거 / 대안
- 채택 이유: Epley는 널리 쓰이고 중간 반복수에서 합리적, 단순·재현 가능. 단일 출처로 일관성·신뢰 확보.
- 검토한 대안: ① **Brzycki** — 고반복에서 보수적이나 표준화 이점 동일, 선택의 문제. ② **사용자 선택형 다중 산식** — 유연하나 PR 비교·AI 강도 일관성 저해 → 초기 비채택(후기 옵션 가능).

## 결과 (Consequences)
- 장점: 일관된 지표·PR·강도 산정, 구현·테스트 단순.
- 트레이드오프: 단일 산식이라 모든 반복 구간에 완벽하진 않음(추정치 한계).
- 되돌림 비용: 낮음 — 산식이 단일 모듈에 캡슐화되면 교체/옵션화 용이.

## 연결
- 영향 SRS/SAD: SAD-003(분석 엔진), SRS-005 · 대체: 없음
