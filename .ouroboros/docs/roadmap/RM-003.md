# RM-003 · P1 쐐기 — Wedge A: AI 지능형 프로그래밍/주기화 (입문 친화)

> **재정렬(2026-07, ADR-020)**: 보고서가 P1 쐐기의 핵심으로 지목(여성·직장인 입문자용 '입문 친화 AI 루틴'). Phase 2 → **Phase 1**. **정합성**: 현재 규칙기반 온디바이스 생성기까지 구현, LLM 보조(ADR-004)·주차 진행 라이프사이클은 부분/미구현 — P1에서 보강.

## 목표
Hevy가 안 하는 "프로그램을 짜주는 코치 레이어"를 만든다. 목표·경력·장비·일정 입력 → 점진적 과부하 프로그램 자동 생성 + 주차별 자동 진행/디로드. **가치 차별화(포맷 아님)**.

## 범위
- AI 프로그램 생성(목표/경력/장비/일정 → 프로그램), 점진적 과부하·자동 주차 진행·디로드.
