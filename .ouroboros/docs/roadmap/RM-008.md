# RM-008 · 소셜 고도화 (인스타 통합) — DM·미디어·스토리·탐색·알림

## 목표
Hevy식 소셜 코어(RM-002: 팔로우·피드·좋아요/댓글·프로필·PR공유) 위에 **인스타식 직접 연결·미디어·발견**을 얹어, 운동 동료·트레이너와 연결되는 소셜 레이어를 완성한다. 핵심 동선: **피드 발견 → 프로필 → DM으로 PT 문의·만남 약속**. '좁은 부족' 책임감 해자(ADR-007)는 유지하고 그 위에 옵트인 공개 레이어를 더한다(ADR-014).

## 범위
- 포함 요구:
  - **URS-010** 운동 동료·트레이너와 직접 연결(DM)되어 PT 문의·만남을 잡고 싶다
  - **SRS-017** 다이렉트 메시지 (1:1·그룹 채팅·이미지·PT문의/만남 조율)
  - **SRS-018** 소셜 그래프·탐색·검색 (팔로우·추천 디스커버리·검색·해시태그/멘션)
  - **SRS-019** 미디어 게시물·스토리 (사진/영상 포스트·24h 스토리)
  - **SRS-020** 통합 알림 센터 (좋아요·댓글·팔로우·DM·멘션·케어)
  - **SRS-007 / SRS-008** (확장): 피드에 미디어 포스트·발견 표면, 프로필에 공개 프로필·DM 진입·PT 문의 CTA
- 설계: SAD-011(소셜·실시간 메시징 백엔드)·SAD-012(미디어 파이프라인·모더레이션)
- 결정: ADR-014(포지셔닝)·ADR-015(메시징 기술)·ADR-016(미디어/CDN)·ADR-017(모더레이션·안전·민감정보)
- 전제: 서버 백엔드·동기 엔진 선행(현재 로컬 전용·동기 0%). 음성/영상 통화는 비범위(후속).

## 일정
- 목표: 2027-03-31 (Planned) — RM-002 소셜 코어(2026-11-30) + 동기 엔진 백엔드 이후 편성.
- 의존: 계정·동기(SRS-006/SAD-004) → 소셜 백엔드(SAD-011) → 미디어(SAD-012).
