# MCP 도구

Ouroboros MCP 서버는 **18개 도구**를 제공합니다. 모든 도구는 Claude Code에서 자동으로 사용 가능합니다.

---

## 메모리 도구

메모리의 저장, 검색, 업데이트, 아카이브, 배치 저장을 담당합니다.

| 도구 | 설명 |
|------|------|
| `memory_store` | 새 메모리 저장 (gotcha, pattern, decision, insight, domain-knowledge, preference) |
| `memory_search` | 메모리 검색 (벡터 유사도 + RRF 리랭킹) |
| `memory_update` | 기존 메모리 내용 수정 |
| `memory_archive` | 메모리를 archived 상태로 전환 (검색에서 제외) |
| `memory_batch_store` | 여러 메모리를 배치로 저장 |

### memory_store

```
memory_store({
  type: "gotcha",
  content: "시장 규모 통계 인용 시 연도/분기 기준을 반드시 명시해야 한다",
  scope: "project",
  tags: ["보고서", "데이터인용"],
  related_files: [".ouroboros/docs/write/2026-03-30-시장분석.md"],
  user_id: "kimjeongmin",
  project_id: "my-project"
})
```

### memory_search

```
memory_search({
  query: "보고서 데이터 인용 주의사항",
  limit: 5,
  user_id: "kimjeongmin",
  project_id: "my-project"
})
```

<Callout type="info">
  `memory_search`는 4축 RRF 리랭킹을 사용합니다. 상세 내용은 [메모리 시스템](/infra/memory)을 참조하세요.
</Callout>

## 지식 그래프 도구

Knowledge Graph의 생성, 조회, 연결, 일반화를 담당합니다.

| 도구 | 설명 |
|------|------|
| `knowledge_store` | KG 노드 생성 (문서 구조, 작성 패턴, 도메인 지식 등) |
| `knowledge_query` | KG 조회 (관계 탐색, 패턴 매칭) |
| `knowledge_link` | 노드 간 관계 생성/수정 |
| `knowledge_distill` | 메모리 → KG 일반화 (DBSCAN 클러스터링) |

### knowledge_store

```
knowledge_store({
  type: "writing_pattern",
  name: "피라미드 구조",
  namespace: "project",
  properties: {
    description: "결론을 먼저 제시하고 근거를 아래에 배치하는 문서 구조",
    applies_to: ["경영진 보고서", "의사결정 문서"]
  }
})
```

### knowledge_query

```
knowledge_query({
  type: "writing_pattern",
  namespace: "project",
  depth: 2
})
```

프로젝트 구조와 메타데이터를 동기화합니다.

### project_sync

```
project_sync({
  project_id: "my-project",

```

## 이벤트 도구

이벤트 기록과 대화 동기화를 담당합니다.

| 도구 | 설명 |
|------|------|
| `event_record` | 단일 이벤트 기록 (도구 호출, 에러 등) |
| `event_sync_conversation` | 대화 전량 동기화 (JSONL) |
| `event_query` | 이벤트 기록 조회 (시간 범위, 필터) |

### event_record

```
event_record({
  type: "tool_call",
  tool: "Edit",
  file: ".ouroboros/docs/write/2026-03-30-보고서.md",
  session_id: "session-123"
})
```

### event_sync_conversation

```
event_sync_conversation({
  session_id: "session-123",
  messages: [ ... ]  // JSONL 형식의 대화 전량
})
```

## 외부 MCP 도구 (필수 활용)

Ouroboros 18개 도구 외에 3개 외부 MCP 도구를 **적극 활용**합니다.

| 도구 | 용도 | 활용 시점 |
|------|------|----------|
| **serena** | 프로젝트 파일 구조 분석 (심볼 탐색, 참조 추적) | 프로젝트 파일을 분석할 때 |
| **context7** | 외부 문서/레퍼런스 최신 자료 조회 | 참고자료 확인, 최신 정보 필요 시 |
| **sequential-thinking** | 복잡한 추론을 단계적으로 분해 | 복잡한 문서 설계, 논리 구조 검증 시 |

## WebSearch 활용 규칙

다음 상황에서는 **반드시** WebSearch를 사용합니다:

- context7에서 문서를 찾을 수 없는 주제
- 내부 분석만으로 충분한 자료를 확보할 수 없을 때
- 최신 통계, 시장 동향, 업계 변경사항 확인 필요 시
- 외부 사례/패턴이 필요한 경우
- context7 MCP 미연결 시 대체 수단
