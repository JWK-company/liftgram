# 대시보드 개요

Ouroboros 대시보드는 메모리, 세션, 지식 그래프, 워크플로우를 웹에서 관리하는 통합 UI입니다.

---

## 접속

| 항목 | 값 |
|------|-----|
| **URL** | `.ouroboros/env/.env`의 `EVENT_LOG_URL` 도메인 |
| **인증** | API 키 (`OUROBOROS_API_KEY`) |

---

## 페이지 구성

### Overview

프로젝트의 전체 현황을 한눈에 파악하는 대시보드 메인 페이지입니다.

| 위젯 | 설명 |
|------|------|
| 메모리 수 | 전체 / active / stale / archived 메모리 수 |
| 활성 세션 | 현재 활성 중인 Claude Code 세션 수 |
| 도구 호출 | 최근 MCP 도구 호출 통계 |
| 최근 활동 | 최근 메모리 저장/검색/아카이브 이력 |

---

### Memories

축적된 메모리를 조회하고 관리하는 페이지입니다.

**기능:**
- 메모리 목록 (타입, scope, 상태별 필터)
- 벡터 유사도 검색 (쿼리 입력 → 관련 메모리 조회)
- 메모리 상세 보기 (내용, 태그, 관련 파일, 접근 이력)
- 상태 변경 (active → archived)
- 메모리 수정/삭제

| 필터 | 옵션 |
|------|------|
| 타입 | gotcha, pattern, decision, insight, domain-knowledge, preference |
| Scope | project, personal |
| 상태 | active, stale_candidate, archived |

---

### Knowledge

Knowledge Graph를 시각화하고 탐색하는 페이지입니다.

**기능:**
- 그래프 시각화 (노드 + 관계 네트워크 뷰)
- 노드 검색 (타입, 이름, 네임스페이스)
- 관계 탐색 (depth 조절)
- 노드 상세 (속성, 연결된 노드 목록)

---

### Workflow

워크플로우 상태와 이력을 모니터링하는 페이지입니다.

**기능:**
- 현재 활성 워크플로우 상태 (스킬, Phase, 등급)
- 워크플로우 이력 (완료된 작업 목록)
- 체크포인트 목록
- 워크플로우 메트릭 (평균 완료 시간, 스킬 사용 빈도)

---

### Sessions

Claude Code 대화 기록을 조회하는 페이지입니다.

**기능:**
- 세션 목록 (시작 시간, 지속 시간, 도구 호출 수)
- 세션 클릭 시 **전체 대화** 조회

세션 상세에서는 다음 메시지 유형을 구분하여 표시합니다:

| 유형 | 아이콘 | 설명 |
|------|--------|------|
| USER | 사용자 | 사용자 입력 |
| ASSISTANT | AI | Claude 응답 |
| TOOL | 도구 | 도구 호출 (파일 경로, 변경 내역) |
| RESULT | 결과 | 도구 실행 결과 |

긴 내용은 **"더보기"** 버튼으로 펼칠 수 있습니다.

<Callout type="info">
  Sessions 데이터는 `conversation-sync` Hook이 ClickHouse에 동기화한 JSONL 대화 전량입니다.
</Callout>

---

### Settings

프로젝트 설정을 관리하는 페이지입니다.

**기능:**
- 프로젝트 정보 조회/수정
- MCP 도구 설정
- 알림 설정
- 데이터 내보내기

---

### Admin

관리자 기능을 제공하는 페이지입니다.

**기능:**
- 라이센스 관리 (발급 / 폐기 / 갱신)
- 사용자 관리
- 시스템 상태 모니터링
- 데이터 백업/복원

<Callout type="warning">
  Admin 페이지는 관리자 권한이 필요합니다.
</Callout>
