# 설치

Ouroboros Document Workflow를 프로젝트에 설치하는 방법을 안내합니다.

---

## 사전 요구사항

| 항목 | 최소 버전 | 설명 |
|------|----------|------|
| **Claude Code** | 최신 | Anthropic CLI (`claude` 명령어) |
| **Git** | 2.30+ | 버전 관리 |
| **curl** | - | 설치 스크립트 다운로드 |
| **bash** | - | Windows: Git Bash 또는 WSL 필요 |
| **Python 3** | 3.8+ | 문서 변환 도구 (`markitdown`) |

<Callout type="info">
  **Windows 사용자:** Git Bash (`git-scm.com`) 또는 WSL에서 설치하세요. PowerShell에서는 실행되지 않습니다.
</Callout>

---

## 설치 방법

<Steps>
### Keycloak 로그인 + 설치 토큰 발급

브라우저에서 설치 페이지에 접속합니다.

```
https://ouro.jungmin.kim/install-doc
```

1. **"Keycloak으로 로그인"** 버튼을 클릭합니다.
2. Keycloak 계정으로 로그인합니다.
3. 로그인 성공 후, **1회용 설치 명령어**가 표시됩니다.

<Callout type="warning">
  설치 토큰은 **10분간 유효**하며, **1회만 사용 가능**합니다. 만료 시 페이지를 새로고침하세요.
</Callout>

### 설치 스크립트 실행

표시된 curl 명령어를 프로젝트 루트 디렉토리에서 실행합니다.

```bash
curl -fsSL 'https://ouro.jungmin.kim/install-doc.sh?token=YOUR_TOKEN' | bash
```

설치 스크립트가 처리하는 항목:

- 기존 `.claude/`, `.ouroboros/`, `.mcp.json`, `CLAUDE.md` 백업
- `.claude/` 디렉토리 (12개 스킬, CLAUDE.md, settings.json)
- `.ouroboros/` 디렉토리 (Hook, context, docs, workflow)
- `.ouroboros/env/.env` (사용자 ID, 프로젝트 ID)
- `.mcp.json` (Keycloak OAuth, 4개 MCP 서버)
- `CLAUDE.md` 워크플로우 가이드 + 문서 읽기 규칙
- `.gitignore` 업데이트

### MCP 서버 연결 (OAuth)

Claude Code에서 MCP 서버에 연결합니다.

```
/mcp → ouroboros → Authenticate
```

브라우저에서 Keycloak 로그인 화면이 열립니다. 로그인하면 자동으로 연결됩니다.

### 문서 변환 도구 설치

PDF, DOCX, XLSX 등 다양한 문서를 읽으려면 `markitdown`을 설치합니다.

```bash
pip install markitdown pymupdf pdfminer.six mammoth openpyxl pandas python-pptx
```

<Callout type="info">
  문서 변환 도구는 선택사항이지만, 원본 자료를 활용하려면 **강력히 권장**합니다.
</Callout>

### 설치 확인

```bash
ls -la .claude/
# CLAUDE.md       — 워크플로우 가이드 (SSoT)
# commands/       — 12개 스킬
# settings.json   — Hook + 권한 설정

ls -la .ouroboros/
# context/        — current.md, state.json
# docs/           — spec, qa, write, knowledge, draft
# hooks/          — Hook 스크립트
# scripts/        — 유틸리티
# workflow/       — 프로토콜, 템플릿
# env/            — .env (서버 연결)
```
</Steps>

## 설치 검증

```bash
make status    # 설정 + Hook + 서버 연결 상태 확인
make doctor    # 전체 건강 진단
make guide     # 시스템 가이드 출력
```

## 시작하기

설치가 완료되면 Claude Code를 실행하고 `/suggest`를 입력하세요.

```bash
claude
```

```
> /suggest
```

현재 프로젝트 상태를 분석하고 다음에 할 작업을 추천합니다.
