# /draft

spec 없이 **1-2페이지 분량의 간단한 문서**를 즉시 작성하는 스킬입니다.

<Callout type="info">
  복잡한 문서는 `/spec` → `/write`를 사용하세요. `/draft`는 빠른 초안 작성 전용입니다.
</Callout>

---

## 언제 사용하나요?

| 적합 | 부적합 |
|------|--------|
| 내부 메모, 이메일 초안 | 10페이지 이상 보고서 |
| 회의록 정리 | 다자 리뷰가 필요한 기획서 |
| 간단한 안내문 | 근거 자료가 많은 백서 |
| 1회성 요약 문서 | 반복 업데이트되는 문서 |

---

## 사용법

```
/draft "문서 내용 설명"
```

### 예시

```
/draft "이번 주 스프린트 회고 메모 — 완료 3건, 이월 1건, 개선점 정리"
```

```
/draft "신규 입사자 온보딩 안내 이메일 — 첫째 주 일정, 필수 도구 목록"
```

---

## 동작 과정

<Steps>
### 요청 분석

문서 유형, 대상 독자, 핵심 메시지를 파악합니다.

### 메모리 검색

관련 메모리(이전 문서 패턴, 용어 정의 등)를 자동 조회합니다.

### 문서 작성

구조화된 초안을 바로 생성합니다. 결과는 `.ouroboros/docs/` 디렉토리에 저장됩니다.

### 다음 단계 안내

필요 시 `/revise`(개선) 또는 `/review`(리뷰)를 안내합니다.
</Steps>

---
